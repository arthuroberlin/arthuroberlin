# Hey, moi c'est Arthur 👋

Développeur **Front-end Senior** basé à Paris, 7 ans à façonner des interfaces sur le web.

Je passe autant de temps dans **Figma** que dans mon éditeur — l'**UX/UI** n'est pas un à-côté chez moi, c'est le moteur. Une bonne interface, c'est une interface dont on ne parle pas : elle laisse passer ce que les gens viennent y faire.

Et surtout, je tiens à ce qu'elle soit **accessible à toutes et tous** : navigation clavier, contrastes, sémantique HTML, lecteurs d'écran — pas une checklist en fin de sprint, mais un réflexe dès la première maquette.

<br/>

## `01` &nbsp; Mon parcours

J'ai bossé sur des terrains très différents :

- **Banque** — interfaces de gestion, dashboards, exigence de rigueur
- **SaaS CSE** — plateformes à fort volume, tests, fiabilité
- **Robotique** — UI temps réel, contraintes hardware
- **E-commerce** — funnels de conversion, performance perçue
- **Médias** — chez [**Konbini**](https://www.konbini.com/), jeux navigateur multijoueurs en React / xState et leur site éditorial en Next.js

Le fil rouge : **du code propre, des composants qui durent, et des interfaces qui respectent l'utilisateur.**
